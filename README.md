## 🚀 安装方法
1. **下载 ZIP**（点右边 `↓ Code` → `Download ZIP`）
2. 解压文件
3. 浏览器打开 `chrome://extensions`
4. 开启 **"开发者模式"**
5. 点击 **"加载已解压的扩展程序"**，选择解压的文件夹
